Sun-Panel ​

Github | Gitee | Docker Hub | B站视频

一个服务器、NAS导航面板、Homepage、浏览器首页。

😎 特点 ​

  • 简洁
  • 局域网内外网链接切换
  • docker部署,对arm系统支持
  • 上手简单，免修改代码
  • 无需连接外部数据库
  • 丰富图标自由搭配（文字图标+svg图标+内置三方图标库）
  • 支持网页内置小窗口打开（部分网站屏蔽此功能）
  • 占用资源小
